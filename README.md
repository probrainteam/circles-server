# Circles-Server

### Express server ``nodejs``

### Gin server ``golang``

## Initialization

express server
```
./run.sh -up express init
```
gin server
```
./run.sh -up gin init
```

## Develop env
express server
```
./run.sh -up express dev
```
gin server
```
./run.sh -up gin dev
```

## Run

express server
```
./run.sh -up express 
```
gin server
```
./run.sh -up gin
```
